# Teste QA jr – Cypress

Este repositório contém testes automatizados de **API** e **E2E (Frontend)** desenvolvidos com **Cypress**.

---

Aplicação
Para execução dos testes, a aplicação deve estar rodando localmente via Docker:

- **Frontend:** http://localhost:5400
- **Backend (API):** http://localhost:8400

---

Pré-requisitos
- Node.js instalado
- Docker em execução
- Aplicação rodando localmente

---

Instalação
Na raiz do projeto, execute:

bash
npm install

---

Executar os testes

Abrir Cypress (modo interativo)

npx cypress open

Executar todos os testes (modo headless)

npx cypress run
